# vue-regex-colorize


## Usage

**in main.js**
```js

import VueRegexColorize from 'vue-regex-colorize'
Vue.use(VueRegexColorize)

```

```html
<VueRegexColorize></VueRegexColorize>
```

import themes from regex-colorize package
``` js
import 'regex-colorize/themes/sweetest.css'
```
